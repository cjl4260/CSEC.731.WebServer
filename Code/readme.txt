Dependencies, compilers/interpreters, and development environments that are necessary to run my code:
 - Python 3.x must be installed

How to run my code:
 Usage: Server.py ip_address port [x509_path] [x509_private_key_path]
 - Examples:
   HTTP Server: sudo python3 HTTP_Server.py 0.0.0.0 80
   HTTPS Server: sudo python3 HTTP_Server.py 0.0.0.0 80 demo.crt demo.key

Note: 
 - Depending on operating system you may need to substitute "python3" for "python" in the terminal command
